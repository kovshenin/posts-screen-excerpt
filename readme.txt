Adds an Excerpt column to the edit posts table. Useful when posting a lot of posts without titles, this will give you an overview of what the post is about.

== Description ==

The plugin's quite simple and straight to the point, doesn't do more than described above. You'll be surprised how short and simple the code is too ;)

== Installation ==

1. Upload the archive to your Wordpress plugins directory, usually `wp-content/plugins/` and unzip the file.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Browse to your All Posts screen and see the new Excerpt column in action

== Frequently Asked Questions ==

= I don't see the new column =

Try using the Screen Options drop-out (top right corner in your admin UI) and see if there's a checkbox for it over there.
